# java-fundamentals-course-smart-solutions

Repositorio del curso introductorio de Java para estudiantes de secundaria.

## Estructura del repositorio
- `/starter-files` – Archivos de inicio para cada lección
- `/completed-examples` – Código fuente completo por lección
- `/scripts` – Scripts de apoyo

## Configuración
No requiere instalación ni configuración local. Todas las prácticas usan el editor en línea "JDoodle"
