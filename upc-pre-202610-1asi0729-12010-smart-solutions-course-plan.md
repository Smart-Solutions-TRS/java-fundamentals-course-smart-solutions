# Fundamentos de Java para Principiantes

## Resumen del Curso

Este curso de 1 hora introduce a estudiantes de secundaria a los fundamentos de la programación con Java, incluyendo programación orientada a objetos. **¡No requiere descargas!** Solo abre tu navegador web.

**Duración total**: ~60 minutos  
**Público Objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación  
**Prerrequisitos**: Ninguno  
**Herramientas necesarias**: **¡Solo tu navegador web!** (Chrome, Firefox, Safari, Edge)  
**📂 Repositorio de Código Fuente**: [https://github.com/Smart-Solutions-TRS/java-fundamentals-course-smart-solutions](https://github.com/Smart-Solutions-TRS/java-fundamentals-course-smart-solutions)

---

## Secuencia de la Lección

### Lección 1: ¿Qué es Java y la Programación? (5 minutos)

- **Descripción**: Aprende qué es la programación y por qué Java es un excelente lenguaje para empezar.
- **Enlace**: [Ver la lección](https://www.youtube.com/watch?v=example-link1)
- **Conclusiones clave**: Java se usa para aplicaciones y juegos; los programas son instrucciones para computadoras.
- **Empieza a programar**: [Abrir JDoodle - Hola Mundo](https://www.jdoodle.com/online-java-compiler/)

### Lección 2: Variables y Tipos de Datos (8 minutos)

- **Descripción**: Aprende a almacenar datos en Java usando variables.
- **Enlace**: [Ver la lección](https://www.youtube.com/watch?v=example-link2)
- **Conclusiones clave**: Usa `int`, `String`, `double`; declara como `int edad = 15;`
- **Práctica**: [Clic para programar](https://www.jdoodle.com/online-java-compiler/) 👈 **¡No es necesario registrarse!**

### Lección 3: Operadores y Expresiones (8 minutos)

- **Descripción**: Aprende a realizar cálculos y comparaciones en Java.
- **Enlace**: [Ver la lección](https://www.youtube.com/watch?v=example-link3)
- **Conclusiones clave**: Operadores aritméticos (`+`, `-`, `*`, `/`); operadores de comparación (`==`, `>`, `<`).
- **Práctica**: [Clic para programar](https://www.jdoodle.com/online-java-compiler/) 👈 **¡Inicio inmediato!**

### Lección 4: Estructuras de Control: Decisiones y Bucles (10 minutos)

- **Descripción**: Toma decisiones con `if-else` y repite tareas con bucles `for` y `while`.
- **Enlace**: [Ver la lección](https://www.youtube.com/watch?v=example-link4)
- **Conclusiones clave**: `if (edad > 12) {}`; `for (int i = 0; i < 5; i++) {}`
- **Práctica**: [Clic para programar](https://www.jdoodle.com/online-java-compiler/) 👈 **¡No necesitas registrarte!**

### Lección 5: Métodos y Entrada/Salida Básica (10 minutos)

- **Descripción**: Crea código reutilizable con métodos y obtén la entrada del usuario con `Scanner`.
- **Enlace**: [Ver la lección](https://www.youtube.com/watch?v=example-link5)
- **Conclusiones clave**: `public static void saludar() {}`; uso de `Scanner` para leer datos del usuario.
- **Práctica**: [Haz clic para practicar](https://www.jdoodle.com/online-java-compiler/) 👈 **¡No necesitas registrarte!**

### Lección 6: Introducción a la POO: Clases y Objetos (12 minutos)

- **Descripción**: Comprende los fundamentos de la programación orientada a objetos creando tus primeras clases y objetos.
- **Enlace**: [Ver la lección](https://www.youtube.com/watch?v=example-link6)
- **Conclusiones clave**: `class Mascota { String nombre; }`; diferencia entre clase y objeto; uso de constructores.
- **Práctica**: [Crea tu primera clase](https://www.jdoodle.com/online-java-compiler/) 👈 **¡Guarda y comparte!**

### Lección 7: Encapsulación y Arreglos (7 minutos)

- **Descripción**: Aplica encapsulación con getters/setters y trabaja con arreglos simples.
- **Enlace**: [Ver la lección](https://www.youtube.com/watch?v=example-link7)
- **Conclusiones clave**: `private String nombre; public String getNombre() {}`; `int[] numeros = new int[5];`
- **Práctica**: [Clic para programar](https://www.jdoodle.com/online-java-compiler/) 👈 **¡Inicio inmediato!**
